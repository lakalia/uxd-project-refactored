The SDETool is a Simple Data Exploration Tool designed to be hooked up to any data set for simple, quick and easy data exploration.

  • Just input the necessary data, hit submit and start exploring!
  • If security is your concern, download the tool to your local machine -- no internet connection required.
The SDETool is a work in progress. Initially it was set up to visualize Hans Rosling's Gapminder population growth data which is the default data set onload. As such, currently, once the form below is actually working, whatever data set is hooked up to the tool must have a Year column and data points by country. The 'Download' tab points to the GitHub repo where this tool is hosted -- click on the green 'Code' button and choose 'Download ZIP' to download the files for the tool.
